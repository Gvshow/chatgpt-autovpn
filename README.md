# V2BridgeLauncher

This is a simple Android launcher app to trigger v2rayNG and connect VPN before launching the target app.

## How to build

1. Use `Android Studio` to open the project and build it.
2. Alternatively, use `Codemagic` for continuous integration:
   - Create a `codemagic.yaml` file in the project root.
   - Push the changes to your GitHub repository.
   - Trigger a build on Codemagic.

## Requirements

- Android SDK 21 or higher
- Gradle 7.4.2 or higher
